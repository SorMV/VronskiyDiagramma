Задача:
Необходимо разработать программу "Зоны близости" на базе технологии swing.
1.	Программа должна строить зоны близости для произвольного множества точек на плоскости (такое разбиение плоскости,
при котором каждая область этого разбиения состоит из точек, более близких к одной из точек исходного множества, чем к
любой другой точке исходного множества. См Диаграмма Вороного)
2.	Программа должна визуализировать исходное множество точек и получившиеся зоны близости.
3.	Программа должна поддерживать добавление точек в набор точек исходного множества с помощью пользовательского
интерфейса
4.	Опционально. Программа должно решать задачу поиска ближайшего соседа через зоны близости
